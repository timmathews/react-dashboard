react-dashboard
===============

Hosting environment for react dashboard components. This is primarily for 
testing purposes and isn't really suitable for actually being a dashboard of 
any kind.

### Usage

```
npm install
npm start
open http://localhost:3000
```

### Dependencies

* React
* Webpack
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
* [babel-loader](https://github.com/babel/babel-loader)
* [react-hot-loader](https://github.com/gaearon/react-hot-loader)
